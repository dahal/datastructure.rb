# Datastructure

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'datastructure'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install datastructure

## Usage

### Queue
```
queue = DataStructure::Queue.new
queue.enque(1)
```
